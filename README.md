# 📖 Morse Code Decoder <a name="about-project"></a>

This is our first project using [Ruby](https://www.ruby-lang.org/en/) for Module 4 - Week 2 of the Full Stack Developer Program offered by [Microverse](https://www.microverse.org/). It's a simple command line program that can decode messages encoded using [Morse Code](https://en.wikipedia.org/wiki/Morse_code).

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need [Ruby](https://www.ruby-lang.org/en/) installed on your machine. You can download the installer for your specific operating system [right here](https://www.ruby-lang.org/en/documentation/installation/).

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/henokkhm/microverse-decode-morse-code.git
```

### Usage

To run the Ruby file, first open the terminal. After changing directories to the *microverse-decode-morse-code* run `ruby morse-code.rb` followed by the message you want to decode.
